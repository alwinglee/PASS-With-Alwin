# Are these valid variable names?

my_name
Name
$house
user message
66
player1

# # # # Solution # # # #

my_name #valid
Name #valid: it's considered bad practice to capitalize the first character of a variable name.
$house #invalid: Variable names cannot start with a symbol.
user message #invalid: Variable names cannot contain spaces.
66 #invalid: A variable name cannot begin with a number.
player1 #valid

# Without altering the values in variables number1 and number2, could you please use type casting to ensure the correct result is displayed?

number1="10"
number2="5"
equation= number1/number2
print("The answer is " + equation)

# # # # Solution # # # #

number1="10"
number2="5"
equation= int(number1)/int(number2)
print("The answer is " + str(equation))
