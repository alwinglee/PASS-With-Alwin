Resources

You’ll find the resources we actively use during our PASS sessions to enrich your learning experience. Please note that these resources are not affiliated with Mohawk College or the PASS Program.

Codellab- IDE

Codellab-IDE is a platform designed to synchronize our IDEs, enhancing collaboration during group work. It was developed by Rodrigo Wong, a former PASS leader.

Python Tutor

Python Tutor is a visual debugger that lets you run your code line by line, helping you pinpoint exactly where errors occur. It's an excellent tool for programmers to identify and resolve issues in their code.
